# Bookstore API

## How to

> Clone the project

```
git clone git@github.com:3ackdoor/Harry-Potter.git
or
git clone https://github.com/3ackdoor/Harry-Potter.git
```

> Open CLI with your code editor (navigate to Harry-Potter folder)

```
code .
```

> Run

```
npm start
```
